This project analyzes user engagement and drop-off rates across different steps of a product funnel using SQL aggregation and filtering.

Analyzed retention and churn patterns of users over a 30-day period using SQL window functions and CASE statements.

CoolTShirts, an innovative apparel shop, is running a bunch of marketing campaigns. In this project, we'll be helping them answer these questions about their campaigns:

  1. How many campaigns and sources does CoolTShirts use and how are they related?
  2. What pages are on their website?
  3. How many first touches is each campaign responsible for?
  4. How many last touches is each campaign responsible for?
  5. How many last touches on the purchase page is each campaign responsible for?
  6. CoolTShirts can re-invest in 5 campaigns. Which should they pick and why?
